# AI 로또 번호 추천기 프로젝트 블루프린트 (v3.0 - AdSense Premium)

## 개요
구글 애드센스 승인을 목표로 하는 프리미엄 디자인 및 고품질 콘텐츠 기반 로또 정보 플랫폼입니다.

## 주요 기능 및 특징
- **번호 생성:** 5가지 전략 기반 알고리즘 제공
- **당첨 기록:** 실시간 당첨 번호 연동 및 시각화
- **전문 블로그:** 10개의 심층 분석 포스트 (SVG 인포그래픽 포함)
- **커뮤니티:** 당첨 후기 공유 게시판
- **프리미엄 UI/UX:** Glassmorphism, 메시 그라데이션, 마이크로 인터랙션 적용

## 진행 계획 및 단계

### 1단계: 디자인 프리미엄화 (진행 중)
- **Visual:** 세련된 다크/라이트 모드 테마 및 OKLCH 기반 생동감 넘치는 컬러 팔레트 적용
- **Layout:** 카드 디자인 고도화 (깊이감 있는 그림자, 글로우 효과)
- **Interaction:** 모든 버튼 및 링크에 부드러운 트랜지션 추가

### 2단계: 애드센스 최적화 (진행 중)
- **Compliance:** 광고 영역 가독성 확보 및 '광고' 레이블 명확화
- **Legal:** 개인정보처리방침에 쿠키 및 광고 관련 조항 보강
- **Navigation:** 사이트맵 수준의 명확한 푸터 및 헤더 내비게이션

### 3단계: 최종 검증 및 배포 (예정)
- 모바일 반응형 완벽 대응 확인
- GitHub 최종 배포 및 승인 신청 준비

## 기술 스택
- **Frontend:** Vanilla JS, Modern CSS (Baseline 2024 features)
- **Design:** Mesh Gradients, CSS Backdrop-filter, OKLCH Color space
- **Data:** External API for lotto history
